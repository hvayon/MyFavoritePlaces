# Python-bot My favorite places
tg: https://t.me/my_favorite_places_bot
## About
his bot allows you to save your favorite places
## How to use
To get started write:

    /start
Then, to add a new location, write

    /add

Enter place name.\
You can add a photo if you like.\
Then add the geolocation of your place.\
After that, a message will appear that everything is saved.\
For command help, type
    
    /help
